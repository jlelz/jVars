# jVars
## Warcraft CVars mod, since 2016

## License
[![License](https://img.shields.io/badge/license-GPL-blue)](LICENSE)

## Why
- Because CVar value standardization did not previously exist
- ...I don't want to look up these settings every time 

## How
Interface > Addons UI

## Platforms
UI works with Classic Era, Classic Wrath and Retail

## FAQ
- I added a lot of CVars but not all
- ...you may request additions
- ...import below is tailored for Mac OS max FPS and has numerous personal settings. You may try it but I would suggest having your own

## Issues
If you're like me and hoping to solve FPS/frame rate issues, you may find that your Graphics Resolution setting in the game is actually causing FPS drops. I would start there, before making other changes
https://github.com/jlelz/jVars/issues
